# Problem 16.53

## Question
A positive point charge \(Q_1 = 2.5 \times 10^{-5}\,\text{C}\) is fixed at the origin of coordinates, and a negative point charge \(Q_2 = -5.0 \times 10^{-6}\,\text{C}\) is fixed on the x axis at \(x = +2.4\,\text{m}\). Find the location(s) along the x axis where the electric field due to these two charges is zero.

## Final answer from source
4.3 m.

## Worked solution
Let the charges be:

- \(Q_1 = +2.5\times 10^{-5}\,\text{C}\) at \(x=0\)
- \(Q_2 = -5.0\times 10^{-6}\,\text{C}\) at \(x=+2.4\,\text{m}\)

We want points on the x axis where the total electric field is zero.

### 1) Check the possible regions
Because the charges have opposite signs, the fields between them point in the same direction, so they cannot cancel between \(x=0\) and \(x=2.4\,\text{m}\).

So cancellation can only occur outside the two charges:

- region I: \(x<0\)
- region III: \(x>2.4\,\text{m}\)

### 2) Set magnitudes equal
At a zero-field point, the magnitudes of the two fields must be equal:

\[
\frac{k|Q_1|}{r_1^2}=\frac{k|Q_2|}{r_2^2}
\]

Cancel \(k\):

\[
\frac{2.5\times 10^{-5}}{r_1^2}=\frac{5.0\times 10^{-6}}{r_2^2}
\]

\[
\frac{2.5\times 10^{-5}}{5.0\times 10^{-6}}=\frac{r_1^2}{r_2^2}
\]

\[
5=\frac{r_1^2}{r_2^2}
\quad\Rightarrow\quad
r_1=\sqrt{5}\,r_2
\]

### 3) Solve in the region \(x>2.4\,\text{m}\)
For a point to the right of both charges:

- distance to \(Q_1\): \(r_1=x\)
- distance to \(Q_2\): \(r_2=x-2.4\)

So

\[
x=\sqrt{5}\,(x-2.4)
\]

\[
x=\sqrt{5}x-2.4\sqrt{5}
\]

\[
(\sqrt{5}-1)x=2.4\sqrt{5}
\]

\[
x=\frac{2.4\sqrt{5}}{\sqrt{5}-1}
\]

Numerically,

\[
x\approx 4.35\,\text{m}
\]

So the location is approximately

\[
\boxed{4.3\,\text{m}}
\]

### 4) Check the region \(x<0\)
If \(x<0\), then \(r_1=-x\) and \(r_2=2.4-x\). The equation becomes

\[
\frac{2.5\times 10^{-5}}{x^2}=\frac{5.0\times 10^{-6}}{(2.4-x)^2}
\]

which leads to

\[
-x=\sqrt{5}(2.4-x)
\]

This gives a negative result inconsistent with the region \(x<0\) when solved, so there is no valid point there.

Thus the only valid zero-field point is at \(x\approx 4.3\,\text{m}\).

## Key concepts used
- Electric field of a point charge: \(E = k|Q|/r^2\)
- Superposition of electric fields
- Opposite-sign charges produce fields that add between them, so cancellation can only occur outside the interval
- Solve by equating field magnitudes and using geometry along the x axis
