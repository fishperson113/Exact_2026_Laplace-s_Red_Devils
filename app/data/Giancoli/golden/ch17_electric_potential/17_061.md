# Problem 17.61

## Question
(I) Write the binary number 1010101010101010 as a decimal number.

## Final answer from source
43,690.

## Worked solution
To convert the binary number $1010101010101010_2 $ to decimal, expand it in powers of 2:

$$ 
1010101010101010_2
=
1\cdot 2^{15}
+0\cdot 2^{14}
+1\cdot 2^{13}
+0\cdot 2^{12}
+1\cdot 2^{11}
+0\cdot 2^{10}
+1\cdot 2^9
+0\cdot 2^8
+1\cdot 2^7
+0\cdot 2^6
+1\cdot 2^5
+0\cdot 2^4
+1\cdot 2^3
+0\cdot 2^2
+1\cdot 2^1
+0\cdot 2^0
 $$

Now add the powers of 2 that have coefficient 1:

$$ 
2^{15}+2^{13}+2^{11}+2^9+2^7+2^5+2^3+2^1
 $$

$$ 
= 32768 + 8192 + 2048 + 512 + 128 + 32 + 8 + 2
 $$

$$ 
= 43690
 $$

So the decimal equivalent is

$$ 
\boxed{43,\!690}
 $$

which agrees with the source answer.

## Key concepts used
- Binary-to-decimal conversion by place value
- Powers of 2 in base-2 notation
- Summing the weighted digits where the binary digit is 1
