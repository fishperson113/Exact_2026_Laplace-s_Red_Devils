# Problem 17.63

## Question
(II) (a) 16-bit sampling provides how many different pos- sible voltages? (b) 24-bit sampling provides how many different possible voltages? (c) For color TV, 3 subpixels, each 8 bits, provides a total of how many different colors?

## Final answer from source
(a) 65,536; (b) 16,777,216; (c) 16,777,216.

## Worked solution
For an \(n\)-bit digital representation, the number of distinct possible values is

\[
2^n
\]

because each bit can be either 0 or 1.

### (a) 16-bit sampling
\[
2^{16} = 65,536
\]

So 16-bit sampling gives **65,536** different possible voltages.

### (b) 24-bit sampling
\[
2^{24} = 16,777,216
\]

So 24-bit sampling gives **16,777,216** different possible voltages.

### (c) Color TV with 3 subpixels, each 8 bits
Each subpixel has

\[
2^8 = 256
\]

possible values. Since there are 3 independent subpixels, the total number of color combinations is

\[
256^3 = (2^8)^3 = 2^{24} = 16,777,216
\]

So the total number of different colors is **16,777,216**.

## Key concepts used
- A binary number with \(n\) bits has \(2^n\) possible combinations.
- Digital sampling resolution determines the number of discrete voltage levels.
- Independent color channels multiply their numbers of levels.
- \(8\)-bit channels correspond to \(256\) levels per channel.
