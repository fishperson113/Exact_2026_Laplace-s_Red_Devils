## Worked solution
This item appears to have incomplete OCR: the resistance value for the dryer element is missing, and the source answer is truncated after “(b)”. Because of that, the numerical result for part (b) cannot be recovered reliably, and the problem should be treated as needing review.

For the conceptual part:

- A resistor does **not** use up charge. Charge is conserved.
- A resistor does **not** use up current. In a series circuit, the same current enters and leaves the resistor.
- What the resistor does is convert electrical energy into thermal energy.

Therefore, for the first multiple-choice question, the correct choice is:

- **(d) Neither (a) nor (b) is true.**

For the quantitative part, if the resistance were known, the method would be:

1. Use Ohm’s law to find the current:
   \[
   I=\frac{V}{R}
   \]
   With \(V=240\ \text{V}\), this would give the current in amperes.

2. Convert time to seconds:
   \[
   50\ \text{min} = 50\times 60 = 3000\ \text{s}
   \]

3. Use
   \[
   Q=It
   \]
   to find the total charge passing through the element in 50 min.

The source answer indicates part (a) is **28 A**, which implies the missing resistance would be about
\[
R=\frac{240}{28}\approx 8.6\ \Omega
\]
matching the visible OCR fragment “8.6”.

Then the charge in 50 min would be
\[
Q=(28\ \text{A})(3000\ \text{s})=8.4\times 10^4\ \text{C}
\]

So the completed answer consistent with the visible source is:
- **(a) 28 A**
- **(b) \(8.4\times 10^4\ \text{C}\)**

- Conservation of charge
- Current is not consumed by a resistor
