# Problem 18.41

## Question
(II) How many 75-W lightbulbs, connected to 120 V as in Fig. 18-20, can be used without blowing a 15-A fuse?

## Final answer from source
24 bulbs.

## Worked solution
For lightbulbs connected in parallel to a 120-V supply, each bulb draws power

\[
P = IV
\]

so the current drawn by one 75-W bulb is

\[
I_{\text{bulb}}=\frac{P}{V}=\frac{75\ \text{W}}{120\ \text{V}}=0.625\ \text{A}.
\]

A 15-A fuse can carry at most 15 A total, so the maximum number \(N\) of such bulbs is

\[
N=\frac{15\ \text{A}}{0.625\ \text{A/bulb}}=24.
\]

Thus, the largest number of 75-W bulbs that can be used without blowing the fuse is

\[
\boxed{24}
\]

which matches the source answer.

## Key concepts used
- Electrical power relation: \(P = IV\)
- Current drawn by a device from a fixed-voltage source
- Total current in parallel circuits is the sum of branch currents
- Fuse rating limits the maximum allowable current
