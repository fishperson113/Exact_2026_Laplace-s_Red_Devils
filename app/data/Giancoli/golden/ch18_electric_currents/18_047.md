# Problem 18.47

## Question
(I) Calculate the peak current in a 2.7-kΩ resistor connected to a 220-V rms ac source.

## Final answer from source
0.12 A.

## Worked solution
For a pure resistor on an ac source, Ohm’s law applies to the rms values:

\[
I_{\text{rms}}=\frac{V_{\text{rms}}}{R}
\]

Given:
- \(V_{\text{rms}} = 220\ \text{V}\)
- \(R = 2.7\ \text{k}\Omega = 2700\ \Omega\)

So,

\[
I_{\text{rms}}=\frac{220}{2700}=0.0815\ \text{A}
\]

The peak current is related to the rms current by

\[
I_{\text{peak}}=\sqrt{2}\, I_{\text{rms}}
\]

Thus,

\[
I_{\text{peak}}=\sqrt{2}(0.0815)\approx 0.115\ \text{A}
\]

Rounding appropriately,

\[
I_{\text{peak}} \approx 0.12\ \text{A}
\]

This matches the source answer.

## Key concepts used
- RMS voltage and current for ac circuits
- Ohm’s law for a resistor: \(I_{\text{rms}}=V_{\text{rms}}/R\)
- Peak-to-rms relation: \(I_{\text{peak}}=\sqrt{2}\,I_{\text{rms}}\)
