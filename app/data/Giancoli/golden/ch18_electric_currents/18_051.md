# Problem 18.51

## Question
(II) An 1800-W arc welder is connected to a 660 -Vrms ac line. Calculate (a) the peak voltage and (b) the peak current.

## Final answer from source
(a) 930 V; (b) 3.9 A.

## Worked solution
For a sinusoidal ac voltage, the rms and peak values are related by

\[
V_{\text{peak}}=\sqrt{2}\,V_{\text{rms}}.
\]

### (a) Peak voltage
Given

\[
V_{\text{rms}}=660\ \text{V},
\]

so

\[
V_{\text{peak}}=\sqrt{2}(660\ \text{V})\approx 933\ \text{V}.
\]

Rounded to two significant figures,

\[
V_{\text{peak}}\approx 930\ \text{V}.
\]

### (b) Peak current
For a resistive load on ac, the average power is

\[
P=V_{\text{rms}} I_{\text{rms}}.
\]

Thus,

\[
I_{\text{rms}}=\frac{P}{V_{\text{rms}}}
=\frac{1800\ \text{W}}{660\ \text{V}}
\approx 2.73\ \text{A}.
\]

The peak current is related to rms current by

\[
I_{\text{peak}}=\sqrt{2}\,I_{\text{rms}}.
\]

So,

\[
I_{\text{peak}}=\sqrt{2}(2.73\ \text{A})\approx 3.86\ \text{A}.
\]

Rounded,

\[
I_{\text{peak}}\approx 3.9\ \text{A}.
\]

These match the source answer.

## Key concepts used
- RMS and peak values for sinusoidal ac: \(V_{\text{peak}}=\sqrt{2}V_{\text{rms}}\), \(I_{\text{peak}}=\sqrt{2}I_{\text{rms}}\)
- Average power relation for a resistive ac load: \(P=V_{\text{rms}}I_{\text{rms}}\)
- Unit consistency and rounding to significant figures
