# Problem 19.9

## Question
(II) Suppose that you have a 9.0-V battery and wish to apply 1.0-(cid:2) a voltage of only 3.5 V. Given an unlimited supply of resistors, how could you connect them to make a “voltage divider” that produces a 3.5-V output for a 9.0-V input? 1.70-k(cid:2)

## Final answer from source
Connect 18 resistors in series; then measure voltage across 7 consecutive series resistors.

## Worked solution
To make a voltage divider, connect resistors in series so the battery voltage is divided in proportion to the resistances.

For identical resistors in series, the same current flows through each resistor, so the voltage drop across each one is the same. Therefore, if there are \(N\) identical resistors in series, the voltage across \(n\) of them is

\[
V_{\text{out}} = \frac{n}{N}V_{\text{in}}.
\]

We want

\[
V_{\text{out}} = 3.5\text{ V}, \qquad V_{\text{in}} = 9.0\text{ V}.
\]

So we need

\[
\frac{V_{\text{out}}}{V_{\text{in}}} = \frac{3.5}{9.0} = \frac{7}{18}.
\]

That means the output must be taken across \(7\) out of every \(18\) equal resistors in the series chain.

Thus, connect 18 identical resistors in series across the 9.0-V battery, and take the output across 7 consecutive resistors. This gives

\[
V_{\text{out}} = \frac{7}{18}(9.0\text{ V}) = 3.5\text{ V}.
\]

So the required voltage divider is exactly:

- 18 equal resistors in series
- output measured across 7 consecutive resistors

which matches the source answer.

## Key concepts used
- Voltage division in a series circuit
- Equal resistors in series share voltage equally
- Output voltage is proportional to the number of resistors across which it is measured
- Ratio \(3.5/9.0 = 7/18\)
