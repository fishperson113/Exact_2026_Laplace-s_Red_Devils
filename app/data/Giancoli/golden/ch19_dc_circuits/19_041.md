# Problem 19.41

## Question
(II) The capacitance of a portion of a circuit is to be reduced from 2900 pF to 1200 pF. What capacitance can be added to the circuit to produce this effect without removing existing circuit elements? Must any existing connections be broken to accomplish this? 7.0-mF

## Final answer from source
yes.

## Worked solution
To reduce the effective capacitance from \(2900\ \text{pF}\) to \(1200\ \text{pF}\) while keeping the existing circuit elements in place, the added capacitor must be connected so that the net capacitance is smaller than the original.

A capacitance can be reduced only by a **series** connection, since for capacitors in series:

\[
\frac{1}{C_{\text{eq}}}=\frac{1}{C_1}+\frac{1}{C_2}
\]

Here,
- \(C_{\text{eq}} = 1200\ \text{pF}\)
- \(C_1 = 2900\ \text{pF}\)
- \(C_2 = ?\)

So,

\[
\frac{1}{1200}=\frac{1}{2900}+\frac{1}{C_2}
\]

\[
\frac{1}{C_2}=\frac{1}{1200}-\frac{1}{2900}
\]

Compute:

\[
\frac{1}{C_2} \approx 8.333\times 10^{-4}-3.448\times 10^{-4}
=4.885\times 10^{-4}\ \text{pF}^{-1}
\]

\[
C_2 \approx \frac{1}{4.885\times 10^{-4}} \approx 2.05\times 10^3\ \text{pF}
\]

So the added capacitance must be about

\[
\boxed{2.1\times 10^3\ \text{pF} \;=\; 2.1\ \text{nF}}
\]

Since this requires placing the new capacitor **in series** with the existing one, **yes**, some existing connections must be broken to insert the capacitor in series.

## Key concepts used
- Capacitance in series reduces the equivalent capacitance.
- Series capacitor formula: \(\frac{1}{C_{\text{eq}}}=\sum \frac{1}{C_i}\).
- To decrease circuit capacitance without removing parts, add a capacitor in series.
- A series connection requires breaking the original connection path.
