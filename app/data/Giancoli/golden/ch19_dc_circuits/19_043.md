# Problem 19.43

## Question
(II) Consider three capacitors, of capacitance 3200 pF, What maximum and minimum 5800 pF, and capacitance can you form from these? How do you make the connection in each case? 0.0100 mF.

## Final answer from source
in parallel, in series.

## Worked solution
The question appears to be asking for the maximum and minimum equivalent capacitance obtainable from three capacitors with capacitances

- $C_1 = 3200\ \text{pF} $
- $C_2 = 5800\ \text{pF} $
- $C_3 = 0.0100\ \text{mF} $

However, the OCR text is incomplete/garbled: the phrase around “What maximum and minimum … capacitance can you form from these?” is missing connecting words, and the unit/value “0.0100 mF” is oddly placed. Because of that, the exact intended list of capacitor values is not fully reliable, so the result should be treated cautiously.

For a set of capacitors:

- The **maximum** equivalent capacitance is obtained by connecting them **in parallel**:
  $$ 
  C_{\max} = C_1 + C_2 + C_3
   $$
- The **minimum** equivalent capacitance is obtained by connecting them **in series**:
  $$ 
  \frac{1}{C_{\min}} = \frac{1}{C_1} + \frac{1}{C_2} + \frac{1}{C_3}
   $$

These are the standard extremal combinations for capacitors.

Since the source answer given is simply **“in parallel, in series,”** the intended response is:

- **Maximum capacitance:** connect all capacitors **in parallel**
- **Minimum capacitance:** connect all capacitors **in series**

## Key concepts used
- Parallel capacitances add directly.
- Series capacitances add reciprocally.
- Maximum equivalent capacitance comes from parallel connection.
- Minimum equivalent capacitance comes from series connection.
- OCR ambiguity can make the numerical values unreliable; the connection rule remains clear.
