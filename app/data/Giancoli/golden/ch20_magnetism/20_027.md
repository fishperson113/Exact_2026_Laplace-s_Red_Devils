# Problem 20.27

## Question
(I) Determine the magnitude and direction of the force between two parallel wires 25 m long and 4.0 cm apart, each carrying 25 A in the same direction.

## Final answer from source
toward other wire.

## Worked solution
For two long parallel wires carrying currents in the same direction, the wires attract each other.

The magnitude of the force between parallel currents is

\[
F=\frac{\mu_0 I_1 I_2 L}{2\pi r}
\]

where:
- \(\mu_0 = 4\pi\times10^{-7}\ \text{N/A}^2\)
- \(I_1 = I_2 = 25\ \text{A}\)
- \(L = 25\ \text{m}\)
- \(r = 4.0\ \text{cm} = 0.040\ \text{m}\)

Substitute:

\[
F=\frac{(4\pi\times10^{-7})(25)(25)(25)}{2\pi(0.040)}
\]

Cancel \(\pi\) and simplify:

\[
F=\frac{2\times10^{-7}(25)(25)(25)}{0.040}
\]

\[
(25)(25)(25)=15625
\]

\[
F=\frac{2\times10^{-7}\times15625}{0.040}
\]

\[
F=\frac{3.125\times10^{-3}}{0.040}=7.8125\times10^{-2}\ \text{N}
\]

So the force magnitude is

\[
F \approx 7.8\times10^{-2}\ \text{N}
\]

Because the currents are in the same direction, each wire is pulled toward the other wire.

## Key concepts used
- Magnetic force between parallel current-carrying wires
- Same-direction currents attract
- Unit conversion: \(4.0\ \text{cm} = 0.040\ \text{m}\)
- Force formula: \(\displaystyle F=\frac{\mu_0 I_1 I_2 L}{2\pi r}\)
