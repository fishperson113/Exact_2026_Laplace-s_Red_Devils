# Problem 20.45

## Question
(I) A 30.0-cm-long solenoid 1.25 cm in diameter is to produce a field of 4.65 mT at its center. How much current should the solenoid carry if it has 935 turns of the wire?

## Final answer from source
1.19 A.

## Worked solution
For a long solenoid, the magnetic field at the center is

\[
B=\mu_0 n I
\]

where

- \(B = 4.65\,\text{mT} = 4.65\times 10^{-3}\,\text{T}\)
- \(\mu_0 = 4\pi\times 10^{-7}\,\text{T·m/A}\)
- \(n = \dfrac{N}{L}\) is the number of turns per unit length
- \(N = 935\)
- \(L = 30.0\,\text{cm} = 0.300\,\text{m}\)

First find \(n\):

\[
n=\frac{935}{0.300}=3116.7\ \text{turns/m}
\]

Now solve for the current:

\[
I=\frac{B}{\mu_0 n}
\]

Substitute the values:

\[
I=\frac{4.65\times 10^{-3}}{(4\pi\times 10^{-7})(3116.7)}
\]

\[
I \approx \frac{4.65\times 10^{-3}}{3.916\times 10^{-3}}
\approx 1.19\ \text{A}
\]

So the solenoid should carry

\[
\boxed{1.19\ \text{A}}
\]

which matches the source answer.

## Key concepts used
- Magnetic field inside an ideal long solenoid: \(B=\mu_0 n I\)
- Turns per unit length: \(n=N/L\)
- Unit conversion: mT to T, cm to m
- Solving algebraically for current from the solenoid field equation
