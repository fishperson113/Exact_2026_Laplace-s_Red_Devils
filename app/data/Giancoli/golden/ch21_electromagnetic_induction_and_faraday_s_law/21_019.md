# Problem 21.19

## Question
(III) The magnetic field perpendicular to a single 13.2-cm- diameter circular loop of copper wire decreases uniformly from 0.670 T to zero. If the wire is 2.25 mm in diameter, how much charge moves past a point in the coil during this operation? 21-5 Generators

## Final answer from source
5.23 C.

## Worked solution
The changing magnetic field produces an induced emf around the loop, which drives a current. The total charge that passes a point is

\[
Q = \int I\,dt = \int \frac{\mathcal{E}}{R}\,dt
\]

For a single loop,

\[
\mathcal{E} = -\frac{d\Phi_B}{dt}
\]

so

\[
Q = \frac{1}{R}\int \left(-\frac{d\Phi_B}{dt}\right)dt
= \frac{\Delta \Phi_B}{R}
\]

Using magnitudes,

\[
Q = \frac{A\,\Delta B}{R}
\]

since the field is perpendicular to the loop and changes from \(0.670\ \text{T}\) to \(0\).

### 1) Loop area
The loop diameter is \(13.2\ \text{cm}\), so

\[
r = 6.60\ \text{cm} = 0.0660\ \text{m}
\]

\[
A = \pi r^2 = \pi(0.0660)^2 = 1.37\times 10^{-2}\ \text{m}^2
\]

### 2) Resistance of the wire
The wire diameter is \(2.25\ \text{mm}\), so its radius is

\[
r_w = 1.125\times 10^{-3}\ \text{m}
\]

Cross-sectional area:

\[
A_w = \pi r_w^2 = \pi(1.125\times 10^{-3})^2
= 3.98\times 10^{-6}\ \text{m}^2
\]

The loop circumference is

\[
L = \pi d = \pi(0.132) = 0.415\ \text{m}
\]

Using the resistivity of copper,
\[
\rho \approx 1.72\times 10^{-8}\ \Omega\cdot\text{m}
\]

the resistance is

\[
R = \rho \frac{L}{A_w}
= (1.72\times 10^{-8})\frac{0.415}{3.98\times 10^{-6}}
\approx 1.79\times 10^{-3}\ \Omega
\]

### 3) Charge transferred
The magnetic flux change is

\[
\Delta \Phi_B = A\,\Delta B
= (1.37\times 10^{-2})(0.670)
= 9.17\times 10^{-3}\ \text{Wb}
\]

Thus,

\[
Q = \frac{\Delta \Phi_B}{R}
= \frac{9.17\times 10^{-3}}{1.79\times 10^{-3}}
\approx 5.23\ \text{C}
\]

So the charge that moves past a point in the coil is

\[
\boxed{5.23\ \text{C}}
\]

## Key concepts used
- Faraday’s law of induction
- Relation between induced emf and current: \(I=\mathcal{E}/R\)
- Total charge passed: \(Q=\int I\,dt\)
- Magnetic flux through a perpendicular loop: \(\Phi_B = BA\)
- Resistance of a wire: \(R=\rho L/A\)
