# Problem 21.33

## Question
(II) A transformer has 330 primary turns and 1240 secondary turns. The input voltage is 120 V and the output current is 15.0 A. What are the output voltage and input current?

## Final answer from source
450 V, 56 A.

## Worked solution
For an ideal transformer,

\[
\frac{V_s}{V_p}=\frac{N_s}{N_p}
\qquad\text{and}\qquad
V_p I_p = V_s I_s
\]

where:
- \(V_p\) = input (primary) voltage
- \(V_s\) = output (secondary) voltage
- \(I_p\) = input (primary) current
- \(I_s\) = output (secondary) current
- \(N_p\) = number of primary turns
- \(N_s\) = number of secondary turns

### 1) Find the output voltage
Given:
\[
N_p=330,\quad N_s=1240,\quad V_p=120\text{ V}
\]

Use the voltage-turns ratio:

\[
V_s = V_p\frac{N_s}{N_p}
\]

\[
V_s = 120\left(\frac{1240}{330}\right)
\approx 120(3.7576)
\approx 451\text{ V}
\]

Rounding to the source answer:

\[
V_s \approx 450\text{ V}
\]

### 2) Find the input current
Given:
\[
I_s = 15.0\text{ A}
\]

For an ideal transformer, power in equals power out:

\[
V_p I_p = V_s I_s
\]

Solve for \(I_p\):

\[
I_p = \frac{V_s I_s}{V_p}
\]

Substitute the values, using \(V_s \approx 450\text{ V}\):

\[
I_p = \frac{(450\text{ V})(15.0\text{ A})}{120\text{ V}}
\]

\[
I_p = \frac{6750}{120}
= 56.25\text{ A}
\]

Rounded to the source answer:

\[
I_p \approx 56\text{ A}
\]

So the output voltage is \(450\text{ V}\), and the input current is \(56\text{ A}\).

## Key concepts used
- Ideal transformer voltage ratio: \(\dfrac{V_s}{V_p}=\dfrac{N_s}{N_p}\)
- Ideal transformer power conservation: \(V_p I_p = V_s I_s\)
- Turn ratio determines voltage step-up or step-down
- Current changes inversely with voltage in an ideal transformer
