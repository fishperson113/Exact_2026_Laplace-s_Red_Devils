# Problem 21.59

## Question
(II) (a) What is the reactance of a well-insulated 0.030-mF capacitor connected to a 2.0-kV (rms) 720-Hz line? (b) What will be the peak value of the current? *21-14 LRC Circuits 36-k(cid:3)

## Final answer from source
(a) (b) 0.38 A.

## Worked solution
For a capacitor connected to an AC source, the capacitive reactance is

\[
X_C=\frac{1}{2\pi f C}.
\]

### (a) Capacitive reactance
Given:
- \(C = 0.030\,\text{mF} = 0.030\times 10^{-3}\,\text{F} = 3.0\times 10^{-5}\,\text{F}\)
- \(f = 720\,\text{Hz}\)

So,

\[
X_C=\frac{1}{2\pi (720)(3.0\times10^{-5})}.
\]

Compute the denominator:

\[
2\pi (720)(3.0\times10^{-5}) \approx 0.136.
\]

Thus,

\[
X_C \approx \frac{1}{0.136} \approx 7.4\,\Omega.
\]

So the reactance is

\[
\boxed{X_C \approx 7.4\,\Omega}.
\]

### (b) Peak current
For a purely capacitive circuit,

\[
I_{\text{peak}}=\frac{V_{\text{peak}}}{X_C}.
\]

The given voltage is rms:

\[
V_{\text{rms}}=2.0\,\text{kV}=2000\,\text{V}.
\]

Convert to peak voltage:

\[
V_{\text{peak}}=\sqrt{2}\,V_{\text{rms}}=\sqrt{2}(2000)\approx 2.83\times10^3\,\text{V}.
\]

Then

\[
I_{\text{peak}}=\frac{2.83\times10^3}{7.4}\approx 3.8\times10^2\,\text{A}.
\]

So the peak current is

\[
\boxed{3.8\times10^2\,\text{A}}.
\]

This is consistent with the source’s listed answer of \(0.38\ \text{A}\) only if the source intended the current in kiliamps, but as written, the calculation from the stated values gives \(3.8\times10^2\ \text{A}\).

## Key concepts used
- Capacitive reactance: \(\displaystyle X_C=\frac{1}{2\pi f C}\)
- RMS to peak conversion: \(\displaystyle V_{\text{peak}}=\sqrt{2}\,V_{\text{rms}}\)
- Ohm’s law for AC capacitive circuits: \(\displaystyle I_{\text{peak}}=\frac{V_{\text{peak}}}{X_C}\)
- Unit conversion: \(0.030\,\text{mF}=3.0\times10^{-5}\,\text{F}\)
