# Problem 22.11

## Question
(I) An EM wave has frequency 7.14 * 1014 Hz. What is its wavelength, and how would we classify it?

## Final answer from source
violet visible light.

## Worked solution
For an electromagnetic wave, the speed of light relation is

\[
c=f\lambda
\]

so the wavelength is

\[
\lambda=\frac{c}{f}
\]

Using \(c=3.00\times10^8\ \text{m/s}\) and \(f=7.14\times10^{14}\ \text{Hz}\),

\[
\lambda=\frac{3.00\times10^8}{7.14\times10^{14}}
\]

\[
\lambda \approx 4.20\times10^{-7}\ \text{m}
\]

Convert to nanometers:

\[
4.20\times10^{-7}\ \text{m}=420\ \text{nm}
\]

A wavelength of about \(420\ \text{nm}\) lies in the violet region of the visible spectrum.

So the wave’s wavelength is \(4.20\times10^{-7}\ \text{m}\) (about \(420\ \text{nm}\)), and it is classified as violet visible light.

## Key concepts used
- Electromagnetic wave relation: \(c=f\lambda\)
- Speed of light in vacuum: \(c=3.00\times10^8\ \text{m/s}\)
- Visible spectrum classification by wavelength
- Unit conversion from meters to nanometers
