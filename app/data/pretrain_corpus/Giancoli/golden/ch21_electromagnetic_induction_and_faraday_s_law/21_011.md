# Problem 21.11

## Question
(II) What is the direction of the induced current in the circu- lar loop due to the current shown in each part of Fig. 21-61? Explain why. I constant I decreasing (a) (c) I decreasing FIGURE 21-61 Problem 11. I increasing (b) (d)

## Final answer from source
(a) 0; (b) clockwise; (c) counterclockwise; (d) clockwise.

## Worked solution
This problem refers to Fig. 21-61, but the figure is not fully available in the prompt. Because the exact geometry and orientation of the wire and loop in each part (a)–(d) are not completely shown, the answer must be taken as a source-constrained result rather than a fully diagram-derived derivation.

Using Faraday’s law and Lenz’s law:

- An induced current appears only if the magnetic flux through the circular loop changes.
- If the source current is constant, the magnetic field and thus the flux through the loop are constant, so there is no induced current.
- If the source current changes, the induced current acts to oppose the change in magnetic flux.

Applying the source answer to the four cases:

- (a) The current is constant, so the flux through the loop does not change. Therefore the induced current is zero.
- (b) The current is increasing, so the magnetic flux through the loop is increasing in the direction set by the external current. The induced current must oppose that increase, which corresponds to a clockwise current.
- (c) The current is decreasing, so the magnetic flux through the loop is decreasing. The induced current must try to maintain the flux, which corresponds to a counterclockwise current.
- (d) The current is increasing in the specific geometry of the figure, and the opposing response of the loop is clockwise.

Thus the directions are exactly those given in the source answer.

## Key concepts used
- Faraday’s law of induction: induced emf arises from changing magnetic flux
- Lenz’s law: induced current opposes the change in flux
- No induced current when magnetic flux is constant
- Right-hand rule to relate current direction and magnetic field direction
- Need for the full figure to determine clockwise/counterclockwise orientation precisely
