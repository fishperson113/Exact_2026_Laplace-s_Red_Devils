# Example 25.6: Using voltmeters and ammeters

## Problem
We move the voltmeter and ammeter in Example 25.5 to different positions in the circuit. What are the readings of the ideal voltmeter and ammeter in the situations shown in (a) Fig. 25.18a and (b) Fig. 25.18b?

## Setup
- In part (a), the voltmeter measures the potential difference between points \(a'\) and \(b'\).
- As in Example 25.5, \(V_{ab} = V_{a'b'}\).
- In part (b), the ideal voltmeter has infinitely large resistance, so if it is placed in the circuit, no current flows in the circuit.
- For the ideal ammeter, the resistance is taken to be zero, so the potential difference across the ammeter terminals is zero.

## Solution
### (a)
The voltmeter now measures the potential difference between points \(a'\) and \(b'\). As in Example 25.5, \(V_{ab} = V_{a'b'}\), so the voltmeter reads the same as in Example 25.5:
\[
V_{a'b'} = 8\ \text{V}.
\]

**Caution: Current in a simple loop**  
As charges move through a resistor, there is a decrease in electric potential energy, but there is no change in the current. The current in a simple loop is the same at every point; it is not “used up” as it moves through a resistor. Hence the ammeter in Fig. 25.17 (“downstream” of the resistor) and the ammeter in Fig. 25.18b (“upstream” of the resistor) both read
\[
I = 2\ \text{A}.
\]

### (b)
There is no current through the ideal voltmeter because it has infinitely large resistance. Since the voltmeter is now part of the circuit, there is no current at all in the circuit, and the ammeter reads
\[
I = 0.
\]

The voltmeter measures the potential difference between points \(b\) and \(b'\). Since the potential difference across the resistor is
\[
V_{a'b'} = IR = 0,
\]
and the potential difference between the ends \(b\) and \(b'\) of the idealized ammeter is also zero, \(V_{bb'}\) is equal to the terminal voltage of the source. As in Conceptual Example 25.4, there is no current, so the terminal voltage equals the emf, and the voltmeter reading is
\[
V_{bb'} = E = 12\ \text{V}.
\]

This example shows that ammeters and voltmeters are circuit elements, too. Moving the voltmeter from the position in Fig. 25.18a to that in Fig. 25.18b makes large changes in the current and potential differences in the circuit. If you want to measure the potential difference between two points in a circuit without disturbing the circuit, use a voltmeter as in Fig. 25.17 or 25.18a, not as in Fig. 25.18b.

## Evaluation
The ideal voltmeter placement in part (a) does not disturb the circuit, so it reads the same potential difference as in Example 25.5. In part (b), the voltmeter’s infinite resistance interrupts the circuit, making the current zero everywhere; then the ammeter reads \(0\), and the voltmeter reads the source emf, \(12\ \text{V}\).

## Key concepts used
- An ideal voltmeter has infinite resistance and must be connected so it does not significantly alter the circuit.
- An ideal ammeter has zero resistance.
- Current in a simple loop is the same at every point in the loop.
- With no current, there is no drop across the external resistor or ideal ammeter, so the terminal voltage equals the emf.
- For the given circuit, \(I = 2\ \text{A}\), \(V_{a'b'} = 8\ \text{V}\), and \(V_{bb'} = 12\ \text{V}\).
