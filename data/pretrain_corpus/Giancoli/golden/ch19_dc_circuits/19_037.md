# Problem 19.37

## Question
(II) A battery for a proposed electric car is to have three hundred 3-V lithium ion cells connected such that the total voltage across all of the cells is 300 V. Describe a possible connection configuration (using series and parallel con- nections) that would meet these battery specifications. 19-5 Capacitors in Series and Parallel

## Final answer from source
3 parallel sets, each with 100 cells in series.

## Worked solution
To obtain a total battery voltage of 300 V from 3-V cells, we need enough cells in series so that their voltages add to 300 V.

1. **Find the number of cells needed in one series string**
   $$ 
   \frac{300\ \text{V}}{3\ \text{V per cell}} = 100
    $$
   So, **100 cells in series** produce 300 V.

2. **Use all 300 cells**
   Since there are 300 cells total, and each series string uses 100 cells:
   $$ 
   \frac{300}{100} = 3
    $$
   So we can make **3 identical series strings**.

3. **Connect the three strings in parallel**
   - Each string has the same voltage: 300 V.
   - Parallel connection keeps the voltage the same across each branch.
   - The capacity/current capability increases, while the total voltage remains 300 V.

Therefore, a suitable configuration is **3 parallel sets, each containing 100 cells connected in series**.

## Key concepts used
- Voltage adds in **series** connections.
- Voltage stays the same across **parallel** branches.
- Total cell count must match the given number of cells.
- A battery pack can combine series and parallel arrangements to reach a desired voltage and capacity.
